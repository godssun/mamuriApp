## Appendix A: 프롬프트 품질 검증 체크리스트

### 말투 검증
- [ ] formal: "~요" 종결어 일관성
- [ ] casual: "~어/아" 종결어 일관성
- [ ] 말투 혼용 없음

### 톤 검증
- [ ] calm: 차분하고 안정적 표현
- [ ] warm: 따뜻하고 공감적 표현
- [ ] cheerful: 밝고 긍정적 표현
- [ ] 톤 혼용 없음

### 조합 검증
- [ ] formal+calm: 정중하고 차분
- [ ] formal+warm: 정중하고 따뜻
- [ ] formal+cheerful: 정중하고 밝음
- [ ] casual+calm: 친근하고 차분
- [ ] casual+warm: 친근하고 따뜻
- [ ] casual+cheerful: 친근하고 밝음

### 위기 상황
- [ ] 안전 메시지 우선
- [ ] 톤 설정 유지 (내용은 안전 우선)
- [ ] 이름 사용 (친근함 유지)
