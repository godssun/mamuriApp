## 1. 기능 개요

### 1.1 문제 정의

**현재 상태**:
- 사용자가 일기를 작성하면 AI가 단 1회의 공감 코멘트를 생성
- AI 응답 후 추가 상호작용 불가능
- 사용자가 AI에게 질문하거나 대화를 이어갈 방법이 없음

**문제점**:
1. **감정적 단절**: 일방향 소통으로 인한 낮은 감정적 만족도
2. **참여도 한계**: AI와의 깊은 교류 부재로 재방문 동기 약화
3. **수익화 기회 부족**: 모든 사용자에게 동일한 경험 제공, 차별화된 가치 제공 불가

### 1.2 솔루션 요약

AI 코멘트에 대한 **사용자 답변 기능**을 추가하여 짧은 대화 스레드 생성 지원.

**핵심 가치**:
- **FREE 티어**: AI와의 기본적 상호작용 경험 (1회 답변)
- **MEDIUM 티어**: 확장된 대화 경험 (일기당 최대 5회 답변)
- **PREMIUM 티어**: 제한 없는 깊은 대화 경험 (무제한 답변)

## 2. 사용자 스토리

### US-001: FREE 사용자 - 기본 대화 경험

**As a** FREE 사용자
**I want to** AI 코멘트에 1회 답변할 수 있기를
**So that** AI와 간단한 대화를 나눌 수 있다

**Acceptance Criteria**:
- [ ] 일기당 AI 코멘트에 1회 답변 가능
- [ ] 답변 제한 도달 시 MEDIUM 구독 안내 표시
- [ ] 답변 후 AI의 재응답 수신
- [ ] 답변 UI에 남은 횟수 표시

### US-002: MEDIUM 사용자 - 확장된 대화

**As a** MEDIUM 구독자 (₩4,900/월)
**I want to** AI와 일기당 최대 5회까지 대화할 수 있기를
**So that** 더 깊은 감정적 교류를 경험할 수 있다

**Acceptance Criteria**:
- [ ] 일기당 최대 5회 답변 가능
- [ ] 답변 횟수 실시간 UI 업데이트
- [ ] 5회 도달 시 PREMIUM 구독 안내
- [ ] 구독 상태 만료 시 FREE 제한으로 자동 전환

### US-003: PREMIUM 사용자 - 무제한 대화

**As a** PREMIUM 구독자 (₩9,900/월)
**I want to** AI와 제한 없이 대화할 수 있기를
**So that** 필요한 만큼 감정적 지원을 받을 수 있다

**Acceptance Criteria**:
- [ ] 일기당 답변 횟수 제한 없음
- [ ] 대화 히스토리 전체 보기 지원
- [ ] 구독 취소 시 grace period 동안 무제한 유지

### US-004: 위기 상황 사용자 예외 처리

**Acceptance Criteria**:
- [ ] `crisis_flag_until` 활성 시 답변 제한 무시
- [ ] 위기 플래그 7일간 유지
- [ ] Safety Event 발생 시 자동 플래그 설정

## 3. 구독 티어 모델

| 기능 | FREE | MEDIUM (₩4,900/월) | PREMIUM (₩9,900/월) |
|------|------|---------------------|---------------------|
| 일기 작성 | 무제한 | 무제한 | 무제한 |
| AI 코멘트 | 월 20건 | 월 100건 | 무제한 |
| AI 대화 (일기당) | **1회 답변** | **5회 답변** | **무제한 답변** |
| AI 동반자 성장 | 레벨 3까지 | 레벨 7까지 | 레벨 10까지 |

### 대화 제한 로직

- 사용자 메시지만 카운트 (AI 응답은 카운트 제외)
- 일기별로 독립적으로 카운트
- 일기 수정해도 카운트 리셋 안 됨
- `crisis_flag_until` 활성 시 모든 제한 무시

## 4. AI 사용량 제한 및 비용 관리

### 4.1 Rate Limiting

| 티어 | 분당 요청 제한 | 시간당 요청 제한 |
|------|----------------|------------------|
| FREE | 5 req/min | 30 req/hour |
| MEDIUM | 10 req/min | 100 req/hour |
| PREMIUM | 20 req/min | 300 req/hour |

### 4.2 비용 예측 (GPT-4o-mini 기준)

- 대화 1회당: ~$0.0005 (₩0.67)
- FREE 사용자 월 비용: ~₩96
- MEDIUM 사용자 월 비용: ~₩480 (수익 ₩4,900, 마진 90.2%)
- PREMIUM 사용자 월 비용: ~₩1,600 (수익 ₩9,900, 마진 83.8%)

## 5. 수익화 영향 분석

### 전환 퍼널

1. **FREE → MEDIUM 전환율**: 7-10% 목표
2. **MEDIUM → PREMIUM 전환율**: 20-30% 목표
3. **주요 전환 트리거**: AI 대화 제한 도달

### 6개월 후 예상 (MAU 5,000명)

```
사용자: FREE 4,250명 / MEDIUM 500명 / PREMIUM 250명
MRR: ₩4,925,000
AI 비용: ₩437,000
순이익: ₩4,488,000 (마진 91.1%)
```

### Analytics 이벤트

```
conversation_started       - 대화 시작
conversation_reply_sent    - 답변 전송
conversation_limit_reached - 제한 도달
upgrade_cta_clicked        - 업그레이드 CTA 클릭
subscription_converted     - 구독 전환
```

## 7. MVP 범위

### MVP (5주)

- Phase 1 (2주): conversation_messages 테이블, API, 티어별 제한
- Phase 2 (1주): 페이월, 업그레이드 모달, 위기 플래그 예외
- Phase 3 (1.5주): 대화 UI, 타이핑 인디케이터, 에러 처리
- Phase 4 (0.5주): Analytics, 모니터링

### Post-MVP

- 음성 대화, 이미지 공유, 대화 요약
- AI 대화 스타일 커스터마이징
- 대화 기반 감정 분석 리포트
